# 🎮 GameMatch - Game Recommendation Website

A fun and interactive website that recommends cool games based on your interests!

## Features

✨ **Interactive Interest Selection**
- Select from 10+ different gaming interests with visual feedback
- Easy-to-use checkbox interface with emoji icons

🎯 **Smart Recommendation Algorithm**
- Matches your selected interests with 25+ games in the database
- Scores games based on how many of your interests they match
- Shows most relevant games first

🎨 **Beautiful User Interface**
- Modern gradient design with smooth animations
- Responsive layout that works on desktop, tablet, and mobile
- Hover effects and smooth transitions

📱 **Fully Responsive**
- Works perfectly on all screen sizes
- Mobile-friendly touch interactions
- Optimized for fast loading

## How It Works

1. **Select Your Interests** - Choose the types of games you like (RPG, Action, Adventure, etc.)
2. **Get Recommendations** - Click "Find My Games" to see personalized recommendations
3. **Explore Results** - Browse through recommended games with descriptions and ratings

## Files Included

- `index.html` - Main HTML structure
- `styles.css` - Beautiful styling and responsive design
- `games-db.js` - Database of 25+ games with genres and information
- `script.js` - Game recommendation algorithm and interactivity
- `README-GameMatch.md` - This file

## How to Use

1. Open `index.html` in your web browser
2. Select one or more interests that appeal to you
3. Click the "Find My Games 🎲" button
4. Browse your personalized game recommendations
5. Each recommendation shows the game title, description, matching genres, and rating

## Customization

You can easily customize this website:

**Add More Games** - Edit `games-db.js` and add new game objects to the `gamesDatabase` array

**Change Colors** - Edit the gradient colors in `styles.css`:
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```

**Add New Interests** - Add new interest categories in both `index.html` and update the algorithm in `script.js` if needed
